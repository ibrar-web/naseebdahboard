# naseebdabord

Web admin dashboard scaffold for the Naseeb AgriTech broker control system.

## Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Redux Toolkit
- React Router
- Axios
- Firebase Messaging
- Socket.io client

## Run

```bash
cd naseebdabord
npm install
npm run dev
```

## Environment

Copy `.env.example` to `.env` and fill in the API, socket, and Firebase values.

## Notes

- This app is for `broker` and `agent` users only.
- `buyer` and `seller` records should be visible as admin reference data, not as self-service users.
